# Rubric registry (mirror)

The canonical source for these rubrics is `daemon/src/rubrics/registry.ts`. The 14 markdown files in this directory are mirrors generated by `pp-daemon dump-rubrics` so the daemon ships self-contained while humans can still read/copy them. Drop a customized rubric at `<project>/.claude/rubrics/<bare-id>.md` to override one for a specific project (loaded via `daemon/src/rubrics/loader.ts`).

- Inspect a rubric body at runtime: `mcp__pp_harness__get_rubric` with the id.
